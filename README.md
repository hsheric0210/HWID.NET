`HWIDmon.NET.exe <view|save|compare> [options]`

`HWIDmon.NET.exe view <-s|--source>`

## Save mode

`HWIDmon.NET.exe save <-o|--out [out-file]> <-s|--source>`

## Compare mode

`HWIDmon.NET.exe compare <-i|--input [in-file]>`

* input (i): The HWID data input file (XML)
